# RoadMap app
Это маленькое приложение на реакте, которое показывает твой прогресс в любом навыке!
### Запуск приложения
В папке, в которой вы хотите включить приложение используйте следующие команды <br>
`git clone https://github.com/username/repository-name.git` <br>
`yarn install` <br>
`yarn start`
